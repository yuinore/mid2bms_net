SoundEngineで音の末尾の無音をフェードアウトしつつ綺麗に削除するスクリプト

(BGM音に対してはこのバッチを適用しないでください。wavの頭に完全な無音がある場合、削除されます。)

[Folder Open]

[Side Insert Silence]
Param0=0
Param1=0.002
Selection=0,-0,-1

[Both Ends Silence Cut]
Parameter1=-90
Parameter2=0
Parameter3=-36
Parameter4=2
Selection=0,-0,-1

[Fade Out]
Selection=-0.002,-0,-1

[Both Ends Silence Cut]
Parameter1=-90
Parameter2=0
Parameter3=-90
Parameter4=0
Selection=0,-0,-1

[Folder Save]



GoldWaveで音の末尾の無音をフェードアウトしつつ綺麗に削除するバッチ



Edit Set Marker/Selection  [T-0] - [T-0]
Edit Insert Silence 0.002
Edit Set Marker/Selection [0] - [T-0]
Edit AutoTrim 0.002s, -36.00dB, Trailing
Edit Set Marker/Selection [T-0.002] - [T-0]
Effect  Fade Out <GoldWave*> Full volume to silence, linear
Edit AutoTrim 0.000s, -90.00dB, Trailing
注：FL Studioでこのソフトを使用する場合のサポートは行いません。
